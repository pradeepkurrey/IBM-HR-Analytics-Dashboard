# IBM HR Analytics Dashboard

## 📌 Project Overview

This project is an interactive HR Analytics Dashboard built using Power BI.
It provides valuable insights into employee demographics, attrition trends, 
salary distribution, job satisfaction, and workforce performance using the IBM HR Analytics dataset.

---

## 🎯 Business Objective

The objective of this dashboard is to help HR managers identify workforce trends, 
monitor employee attrition, and support data-driven HR decision-making.

---

## 📊 Dashboard Pages

- Executive Overview
- Attrition Analysis
- Employee Demographics
- Business Insights

---

## 📈 Key KPIs

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Salary
- Average Age
- Average Experience
- Average Job Satisfaction

---

## 💡 Key Insights

- Sales department has the highest attrition.
- Employees working overtime are more likely to leave the company.
- Most employees belong to the 25–35 age group.
- Active employees account for the majority of the workforce.
- Average salary and experience vary significantly across departments.

---

- Microsoft Excel
